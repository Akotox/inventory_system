# Costech Landing Page Design System

- **Theme**: Dark Mode / Futuristic / Cyberpunk-lite
- **Colors**:
  - Background: #050505 (Deep Black)
  - Primary: #0066FF (Electric Blue)
  - Secondary: #00F2FF (Neon Cyan)
  - Text: #FFFFFF (White) / #94A3B8 (Slate-400)
- **Typography**: 
  - Headings: Orbitron (or fallback sans-serif with high letter-spacing)
  - Body: Inter / System Sans
- **Styles**:
  - Glassmorphism: Background blur (12px), semi-transparent borders.
  - Borders: Thin, glowing cyan/blue gradients.
  - Buttons: Rounded-full, high-intensity glow on hover.
- **Animations**:
  - Subtle pulse on interactive elements.
  - Smooth fade-ins for sections.
  - Glitch effect on headers.
